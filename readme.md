Latar Belakang
Pertumbuhan jumlah mahasiswa dan pekerja perantauan setiap tahunnya meningkatkan kebutuhan akan hunian sementara seperti kos-kosan. Namun, proses pengelolaan kos-kosan di banyak tempat masih dilakukan secara manual, mulai dari pencatatan penghuni, pembayaran sewa, hingga ketersediaan kamar. Permasalahan yang sering terjadi antara lain:
- Kesulitan memantau kamar kosong dan terisi
- Pencatatan pembayaran yang tidak terorganisir
- Minimnya transparansi data penghuni
- Risiko kehilangan atau kesalahan data
Berdasarkan permasalahan tersebut, dikembangkan Sistem Informasi Manajemen Kos-Kosan, yaitu aplikasi berbasis web yang bertujuan untuk membantu pemilik kos dalam mengelola data kamar, penghuni, dan transaksi pembayaran secara digital dan terstruktur.

Tujuan Pengembangan
- Meningkatkan efisiensi pengelolaan kos-kosan
- Mengurangi kesalahan pencatatan manual
- Mempermudah monitoring status kamar
- Menyediakan sistem pencatatan pembayaran yang terintegrasi

Panduan Penggunaan Aplikasi
Panduan ini menjelaskan langkah-langkah penggunaan Sistem Informasi Manajemen Kos-Kosan dari sisi Admin/Pengelola.

1) Login ke Sistem
- Buka website melalui browser.
- Masukkan username dan password pada halaman login.
- Klik tombol Login.
- Jika data benar, sistem akan mengarahkan ke halaman Dashboard.
Tujuan fitur ini adalah untuk menjaga keamanan data dan membatasi akses hanya kepada admin yang berwenang.

2) Menggunakan Dashboard
Setelah login, pengguna akan melihat halaman Dashboard yang berisi:
- Jumlah kamar yang terisi
- Jumlah kamar kosong
- Total penghuni aktif
- Ringkasan status pembayaran
Dashboard berfungsi sebagai pusat monitoring untuk melihat kondisi kos secara cepat dan efisien.

3) Mengelola Data Kamar
Untuk menambahkan atau mengubah data kamar:
- Klik menu Data Kamar.
- Klik tombol Tambah Kamar untuk menambahkan kamar baru.
Isi informasi seperti:
- Nomor kamar
- Harga sewa
- Fasilitas
- Status kamar
- Klik Simpan.
Untuk mengedit atau menghapus data kamar, gunakan tombol Edit atau Hapus pada daftar kamar.

4) Mengelola Data Penghuni
Untuk mengelola penghuni kos:
- Klik menu Data Penghuni.
- Klik Tambah Penghuni.
Isi data penghuni seperti: 
- Nama lengkap
- Nomor telepon
- Nomor kamar
- Tanggal mulai sewa
- Klik Simpan.
Data penghuni akan otomatis terhubung dengan status kamar yang ditempati.

5) Mencatat Pembayaran
Untuk mencatat pembayaran sewa:
- Klik menu Pembayaran.
- Pilih nama penghuni.
Masukkan:
- Tanggal pembayaran
- Nominal pembayaran
- Periode sewa
- Klik Simpan.
Sistem akan memperbarui status pembayaran menjadi Lunas atau Belum Lunas sesuai data yang dimasukkan.

6) Melihat dan Mengunduh Laporan
Grafik Pemasukan per Bulan (Status Lunas)
Fitur Laporan Keuangan pada sistem ini menyajikan visualisasi data dalam bentuk grafik pemasukan bulanan yang dihitung berdasarkan transaksi pembayaran dengan status Lunas.

7) Melihat Peraturan Kost
- Klik menu Peraturan Kost.
- Sistem akan menampilkan daftar tata tertib yang berlaku.
- Admin dapat menambah atau mengedit peraturan sesuai kebutuhan.
Fitur ini bertujuan untuk meningkatkan transparansi dan kedisiplinan penghuni.

Ringkasan Fitur
Sistem Informasi Manajemen Kos-Kosan ini dirancang untuk membantu pemilik atau pengelola kos dalam mengelola operasional secara terstruktur. Berikut adalah fitur utama yang tersedia:

Dashboard
Dashboard merupakan halaman utama yang menampilkan ringkasan informasi penting secara real-time, seperti:
- Total Kamar (angka)
- Kamar Kosong (angka)
- Total Penghuni (angka)
- Tanggal dan waktu saat ini
Fitur ini membantu admin dalam melakukan monitoring kondisi kos secara cepat tanpa harus membuka menu satu per satu.

Data Kamar
Menu ini digunakan untuk mengelola seluruh informasi terkait kamar kos, meliputi:
- Nomor / kode kamar
- Harga sewa per bulan
- Fasilitas kamar
- Status kamar (Kosong / Terisi)
Admin dapat menambahkan, mengedit, dan menghapus data kamar sesuai kebutuhan.

Data Penghuni
Fitur ini digunakan untuk menyimpan dan mengelola informasi penghuni kos, seperti:
- Nama lengkap penghuni
- Nomor identitas / NIK
- Nomor telepon
- Tanggal mulai sewa
- Durasi sewa
Data ini membantu pengelola dalam melakukan pendataan penghuni secara rapi dan terstruktur.

Pembayaran
Menu pembayaran berfungsi untuk mencatat transaksi pembayaran sewa kamar, yang mencakup:
- Nama penghuni
- Nomor kamar
- Tanggal pembayaran
- Nominal pembayaran
- Status pembayaran (Lunas / Belum Lunas)
Fitur ini memudahkan admin dalam memantau penghuni yang sudah melakukan pembayaran dan yang belum membayar.

Laporan keuangan
Grafik Pemasukan per Bulan (Status Lunas)
Fitur Laporan Keuangan pada sistem ini menyajikan visualisasi data dalam bentuk grafik pemasukan bulanan yang dihitung berdasarkan transaksi pembayaran dengan status Lunas.

Tujuan Fitur
Fitur ini bertujuan untuk:
- Menampilkan total pemasukan kos setiap bulan
- Membantu pemilik kos dalam memantau arus kas
- Menjadi dasar evaluasi kinerja keuangan
- Mendukung pengambilan keputusan strategis

Peraturan Kost
Menu ini berisi daftar peraturan atau tata tertib yang berlaku di lingkungan kos, seperti:
- Kewajiban Penghuni
- Peraturan Bertamu
- Larangan Keras (Sanksi Drop Out)
- Disertai juga dengan Denda dan Sanksi Pelanggaran
Fitur ini bertujuan untuk meningkatkan transparansi dan kedisiplinan penghuni kos.
